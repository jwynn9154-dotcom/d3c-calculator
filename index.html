1. Patient demographics

D3C uses Cockcroft-Gault-derived production rate (so weight matters) and Vd = 0.6 × weight. The eGFR transformation cancels weight out, but D3C* (CrCl mL/min) keeps it. Height is used to de-index CKD-EPI back to raw mL/min for side-by-side comparison.

2. Transplant timing & volume status

Any creatinine drawn within 24 hours of the transplant time is flagged as low-confidence because of perioperative fluid shifts. If the patient is volume-overloaded, enter the current weight; Vd will pick up the extra fluid as pure water (production rate stays anchored to dry weight).

4. Trajectory

How to read this

D3C* mL/min is the dynamic creatinine clearance. It's the number to use for drug dose adjustments while creatinine is still falling. Compare against the static CKD-EPI column: in the first 2 to 3 days post-transplant, D3C* should be substantially higher, which is the whole point of the dynamic formula.

D3C eGFR (mL/min/1.73 m²) is the same calculation rewritten to slot directly into the CKD-EPI framework. Use this when you want a single number that means the same thing as the eGFR the patient will have at steady state. Weight cancels out of this form (per Pieters), so it doesn't depend on dry weight accuracy.

Once daily Cr change drops below ~5%, you've effectively reached steady state and standard CKD-EPI is fine. The dynamic and static columns should converge.

If creatinine is rising (negative D3C* trend), the formula still computes correctly, but the constant-GFR-over-the-interval assumption is shakier. Tighten the sampling interval (q8 to q12h) for allograft AKI workup. The "rising" flag marks intervals where Cr increased by more than 10%.

Formulas reference

D3C* (unindexed CrCl in mL/min)
D3C* = [Pcr − Vd × (ΔC/ΔT)] / C_geomean Pcr (mg/h) = (140 − age) × weight × sexFactor / 120 Vd (dL) = 6 × weight (0.6 L/kg, in dL since C is in mg/dL) ΔC (mg/dL) = C2 − C1 ΔT (h) = hours between draws C_geomean = √(C1 × C2) in mg/dL sexFactor = 1 for male, 0.85 for female Numerator units: mg/h Denominator units: mg/dL Result: dL/h × (100 mL/dL ÷ 60 min/h) = mL/min
D3C eGFR (mL/min/1.73 m²) via back-calculated Css
Css (μmol/L) = C_geomean × (140 − A) × k / [k × (140 − A) − 0.6 × (ΔC / ΔT)] k = 0.07362 (male) k = 0.06258 (female) C_geomean and ΔC are in μmol/L (mg/dL × 88.4) ΔT in hours Then run Css (mg/dL) through CKD-EPI 2021 to get D3C eGFR. Weight cancels out of this form.
CKD-EPI 2021 (race-free)
eGFR = 142 × min(PCr/κ, 1)^α × max(PCr/κ, 1)^(−1.200) × 0.9938^age × (1.012 if female) Female: κ = 0.7, α = −0.241 Male: κ = 0.9, α = −0.302
Inker LA et al. N Engl J Med 385:1737-1749, 2021.
Mosteller BSA
BSA (m²) = √(height_cm × weight_kg / 3600)
